#开发工具链（国内项目）

##前端开发工具：

  1. sublime(轻便)
  2. Webstorm（全面，包括各种插件，对node支持较好）
  3. git/svn（gitlab）
  4. npm
  5. 对比工具：Windows：beyondcompare
    osx:Diffmerge(free or professional)
  6. markdown
  7. 命令行:
    • Windows：git bash 好处：可以在任意文件夹右键
    • OSX：iTerm
  8. HBuilder:专为H5设计
  9. pyhton:一些重复工作
